# despicable_characters

自我描述的小巧精美的App

## 如何启动

克隆工程然后 运行 flutter run 

## FlutterSDK 版本

>=2.3.0 <3.0.0

## 第三方开源库

after_layout 版本为: ^1.0.7+2
